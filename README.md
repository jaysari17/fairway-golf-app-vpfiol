## Features

- **Worldwide Course Search**: Search for any golf course globally using real-time API
- **Course Logging**: Log rounds at any golf course worldwide
- **Rating System**: Rate courses on a 1-100 scale with detailed reviews
- **Social Feed**: Share your rounds and see what friends are playing
- **Profile**: Build your golf identity with stats, badges, and achievements
- **Discovery**: Find new courses based on your preferences

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the app and tap the "+" button to search for any golf course worldwide!
